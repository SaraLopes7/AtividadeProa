Como foi minhas férias

Assim que terminou o primeiro período da faculdade, eu resolvi que minhas férias seriam na base dos estudos.
Dessa maneira, construi dois quadros de horários para o mês de férias que eu teria

A primeira semana

Cada sigla dessa primeira semana se refere a um módulo do bootcamp que eu comecei.
O Bootcamp tinha 62 horas, e eu inocente, jurava que ia terminar ele em 1 semana, como estava no meu cronograma.

Os módulos

Esses foram todos módulos que eu estudei nesse Bootcamp.

  1. Introdução a Programação e Pensamento Computacional
  2. Introdução ao Git e o GitHub
  3. Dominando IDEs Java
  4. Variáveis, Tipos de Dados e Operadores matemáticos
  5. Entendendo Métodos Java
  6. Lógica Condicional e controle de Fluxos em Java
  7. Estrutura de Repetição e Arrays em Java
  8. Trabalhando com collections Java
  9. Programação Orientada Objetos
  10. Debugging Java
  11. Tratamento de Excessões em Java
  12. Gerenciamento de Dependências e Build em Java com Maven
  13. Principais Protocolos de Comunicação na Internet
  14. Introdução ao Spring FrameWork
